# Portal de Estudos

Sistema web para gerenciamento e consulta de conteúdos de estudo.

## Tecnologias

- PHP 8.3
- Laravel 11
- JavaScript
- Bootstrap 5
- Firebird 5
- Docker
- Git

## Objetivos

- Aprender desenvolvimento profissional
- Aplicar Engenharia de Software
- Utilizar Docker
- Utilizar Testes Automatizados
- Utilizar Monitoramento
- Preparar sistema para comercialização futura
